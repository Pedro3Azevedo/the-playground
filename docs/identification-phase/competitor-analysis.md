# Competitor Analysis

In Portugal there are some platforms to book courts for several sports, some including basketball.
For example, AirCourts [reference](https://www.aircourts.com/). In this platform, users can choose a sport, the location and a date and see which private courts are available, book it and after playing, the user pays.
There is also the Playtomic [reference](https://playtomic.com/) platform, that is more focused for tennis and padel court booking.
Then, there is Tiesports [reference](https://tiesports.com/quem-somos/) a portuguese startup to platform to manage tennis and padel clubs and academies. Users can book courts, classes and go to tournaments. 
There are other platforms dedicated to club management, but, in Portugal there are not any for Basketball players to challenge each other in casual game or to check public court conditions or availability.
However, there is a platform called _meetup_ where people can meet new people in different occasions and types of events. There is a group called "Lisbon Basketball Club" [reference](https://www.meetup.com/lisbon-basketball-meetup/) where events are created, for example "Wednesday Indoor Basketball" and members of the group can join and play.

Internationally there is a platform endorsed by FIBA called _Courts of the World_ [reference](https://www.courtsoftheworld.com/) where users can look for courts in a certain area.
There is the Pickup app [reference](https://www.pickupgames.app/), that has similar feature has the goal of this project, extending for more sports. There is not much information about the apps creating, it has a small number of downloads in the Play Store and by some social media posts, looks like it is from Hong Kong and has more use there.
